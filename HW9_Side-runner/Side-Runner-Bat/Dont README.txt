SELENIUM-SIDE-RUNNER README!

Этот bat файл помогает установить необходимые компоненты для запуска .side тест кейсов Selenium IDE 
и запускать их!

Рекомендуется запускать его из Командной строки, а не двойным кликом. В таком случае после выпопления 
тест-комплекта окно программы не закроется и будет доступен отчёт для просмотра.

После запуска нужно перейти в меню Установки и последовательно пройти все пункты. После исполнения 
некоторых пунктов программа переходит в режим Командной строки, в таком случае следует ввести имя 
этого файла Side-runner.bat, чтобы запустить его снова.

Запуск side проекто происходит в соответсвующем меню. Нужно ввести цифру желаемого браузера нажать ВВОД
и затем ввести имя проекта, например 1.side, если он находится в той же папке что и этот bat файл,
либо ввести адрес до желаемого файла C:\temp\1.side, нажать ВВОД. 

Этот bat прописывает значение set PATH=%PATH%;C:\temp\driver   только пока исполняется сам этот файл.
В эту же папку он копирует вебдрайверы браузеров во время установки  "Скопировать webdrivers  .exe"
из своей соседней папки \driver.

Опера работает с драйвером хрома, так что фактически запускается хром. Пока так:) 
Впрочем нигде в документаци к Selenium IDE нет ни слова про оперу. Просто через Selenium - можно.
