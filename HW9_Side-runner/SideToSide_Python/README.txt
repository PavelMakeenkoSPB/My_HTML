Программа "Side-to-Side.py" помогает сливать вместе тест-кейсы из двух проектов side в один!

Чтобы всё получилось успешно нужно:

1. Создать два side  проекта в Selenium IDE без добавления тест-кейсов в Test Suit!
2. Скопировать их в папку с этой программой
3. Переименовать один из файлов в "A.side" , а другой в "B.side" 
4. Запустить программу
5. Получить файл Final.side

Далее его можно открыть в Selenium IDE, добавить кейсы в сьюит и в его настройках установить
чек-бокс Run in parallel для запуска одновременно разными браузерами и сохранить проект.

ВАЖНО! Файлы, которые подверглись переименованию будут удалены из текущей папки. Используйте копии!
